# Pnlo 0.0.1 (Purple Neon Light Object) 

Pnlo配置文件代码语法简化比较好，再用这样行吧。支持压缩代码、更多嵌套。

## 代码语法
```kt
"Object1" = "obj-1";
"Object3" = "obj-3";

Object2 = "obj-2";
Object4 = "obj-4";

Time = 2207;

OFF = false;
ON = true;

```

## 注释风格完全一致
```kt
/**/ //
```

### 对象定义
```kt
// 一般形式:
[obj]
   top = 90;
~
```
