# Pnlo 0.0.1 (Purple Neon Light Object) 

Pnlo配置文件代码语法简化比较好，再用这样行吧。支持压缩代码、更多嵌套。

## 代码语法
```kt

// Pnlo This is a comment 
"NSO" = "NeonScript Object";
"version" = "1.0";

Object1 = "obj-1";
Object2 = "obj-2";

"Object3" = "obj-3";
"Object4" = "obj-4";

main_obj_ = "Main Obj";
Time = 2277;
OFF = false;
ON = true;

// 注释风格完全一致
/**/
//

// 对象定义， 一般形式:
[obj]
   yes = true;
   num = 90;
~
```
